[[bean_validation]]
==== Bean Validation

Bean validation is an optional mechanism that provides uniform validation of data on the middleware, in <<gui_framework,Generic UI>> and {rest_man_url}[REST API]. It is based on the JSR 380 - Bean Validation 2.0 and its reference implementation: https://docs.jboss.org/hibernate/stable/validator/reference/en-US/html_single/?v=5.3[Hibernate Validator].
