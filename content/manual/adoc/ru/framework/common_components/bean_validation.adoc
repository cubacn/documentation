[[bean_validation]]
==== Bean Validation

Bean Validation - опциональный механизм, обеспечивающий единообразную валидацию данных на среднем слое, в <<gui_framework,Generic UI>> и в {rest_man_url}[REST API]. Он основан на спецификации JSR 380 - Bean Validation 2.0 и ее референсной имплементации: https://docs.jboss.org/hibernate/stable/validator/reference/en-US/html_single/?v=5.3[Hibernate Validator].
