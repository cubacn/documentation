[[serverInfoMBean]]
====== ServerInfoMBean

http://files.cuba-platform.com/javadoc/cuba/7.1/com/haulmont/cuba/core/jmx/ServerInfoMBean.html[ServerInfoMBean] предоставляет общую информацию о данном блоке Middleware: номер и дату сборки, <<serverId,идентификатор сервера>>.

JMX ObjectName: `app-core.cuba:type=ServerInfo`
